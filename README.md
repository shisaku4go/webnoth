# Webnoth
Welcome to **Webnoth**! This is a fan-made, experimental monorepo dedicated to bringing elements of *The Battle for Wesnoth* into the browser using modern web technologies.

While the ultimate, long-term goal is a full web port of *The Battle for Wesnoth*, the game's massive codebase and long history make a direct port/parallel development a massive challenge. Instead, this repository serves as a playground to build smaller, functional subsets, clones, and experimental tools.

---

## 🚀 Project Vision & Motivation

This project stems from a few key motivations:
- **Play Wesnoth in the Browser:** Making the Wesnoth experience instantly accessible to anyone, anywhere, without needing a desktop installation.
- **AI Agent Experiments:** Exploring new ways to play and interact with the game using AI agents.
- **Web-First Approach:** Leveraging TypeScript and modern web ecosystems rather than traditional downloadable client architectures.

By building lightweight web tools, both long-time Wesnoth fans and newcomers can easily try out these experiments directly from their browsers.
